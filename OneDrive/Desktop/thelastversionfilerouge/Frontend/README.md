# Frontend (Vue 3)

This folder is the standalone Vue frontend for the Laravel backend in `../Src`.

## Run

1. Install dependencies:
   npm install
2. Copy env:
   cp .env.example .env
3. Start dev server:
   npm run dev

## Backend

Run Laravel backend from `../Src`:

- php artisan serve

Default API URL used by frontend:

- VITE_API_BASE_URL=http://127.0.0.1:8000/api
